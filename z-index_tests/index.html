Using z-index in CSS

What is z-index?

Z-index defines the order of which elements on the page are stacked on top of each other

    By default, elements are shown in this order:
  1. Background and root elements.
  2. Non-positioned elements in order of appearance on the HTML.
  3. Positioned elements in order of appearance on the HTML.
    Div 1: Position static
    Div 2: Position relative
    z-index: 9001
    Div 3: Position relative
